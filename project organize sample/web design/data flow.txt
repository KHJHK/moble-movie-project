1. 영화 선택 - Movie table의 key값 비교
2. 영화관(장소) 선택 - 스케줄 보고 날짜, 시간 받아오기 / 영화가 해당 영화관에 없으면 표시 X 
- 영화 key값으로 schedule 보기 → 영화가 있는 Theater 검색 → Theater의 Cinema 검색 → Cinema 정보 화면에 출력

3. 날짜 선택 - 영화관(Theater의 외래키 값 Cinema 키 값을 통해 검색) 및 영화 정보가 일치하는 데이터의 날짜들을 화면에 출력
- 시간이 다른 영화면 날짜가 같아도 전부 출력됨 - 시간이 같아도 무시하고 한 줄로 출력할 필요가 있음
4. 시간 선택 - 날짜 정보가 일치하는 데이터들의 시간값들을 화면에 출력
5. 좌석 선택 - schedule 키값과 일치하는 Seat들 중 선택된 Seat의 키 값과 일치하는 데이터 선택 / 좌석 선택 여부 데이터 변경
6. 예매 선택 후 저장 - Seat에서 골라진 데이터 및 Member 데이터를 Pick 테이블에 저장
